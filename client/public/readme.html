Social Network Project Readme

Social Network Name:

SportyNet

Instructions for starting the server:

Inside the server folder run : "node app.js"
Inside the client folder run : "npm start"
Browse to http://localhost:3000/login and you can login or register a new user.

Server Routes Supported

  • /admin/remove-user?username=:username (DELETE): Remove a user
  • /userLogin/register (POST): User registration
  • /userLogin/login (POST): User login
  • /userLogin/searchUser?username=:username (GET): Search for users by username
  • /userLogin/logout (POST): User logout
  • /following/followers?userID=:userID (GET): Get followers of a user
  • /following/follow (POST): Follow a user
  • /following/unfollow (POST): Unfollow a user
  • /following/allUsers (GET): Fetch all users
  • /following/checkFollowing/:userID/:targetUser (GET): Check if one user follows another
  • /post/createPost (POST): Create a new post
  • /post/editPost/:postId (PUT): Edit an existing post
  • /post/likePost (PUT): Like a post
  • /post/unlikePost (PUT): Unlike a post
  • /post/likeCount/:postId (GET): Get like count for a post
  • /admin/activity-log (GET): Get activity log (Admin only)
  • /admin/enable-disable/pages (PUT): Enable/disable additional pages (Admin only)
  • /admin/enable-disable/features (PUT): Enable/disable additional features (Admin only)
  • /admin/pages (GET): Get pages info (Admin only)
  • /admin/features (GET): Get features info (Admin only)
  • /about (GET): About Us page
  • /contact (GET): Contact Us page
