Server Routes Supported

  • /register (POST): User registration
  • /login (POST): User login
  • /follow (POST): Follow/unfollow users
  • /post (POST): Create a new post
  • /like (POST): Like a post
  • /post/:postId (PUT): Edit an existing post
  • /users/search/:suffix (GET): Search for users by suffix
  • /admin/activity-log (GET): Get activity log (Admin only)
  • /admin/enable-disable/pages (PUT): Enable/disable additional pages (Admin only)
  • /admin/enable-disable/features (PUT): Enable/disable additional features (Admin only)
  • /admin/remove-user/:username (DELETE): Remove a user (Admin only)
  • /about (GET): About Us page
  • /contact (GET): Contact Us page
